# Low Power VLSI Clock Gating

## Overview
This project demonstrates the implementation of clock gating in Verilog HDL to reduce dynamic power consumption in digital VLSI circuits.

## Tools Used
-Verilog HDL
-Xilinx Vivado
-Vivado Simulator
